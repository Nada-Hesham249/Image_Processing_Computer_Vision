# 🧠 Image processing Desktop Application 

A **desktop-based computer vision application** built using **Python, PyQt, and OpenCV**.  
This project provides an interactive GUI for applying and visualizing core image processing techniques in real-time.

---

## ✨ Key Features

### 🖼️ Image Loader
- Load RGB and Grayscale images directly from the GUI
- Real-time image preview inside the application

---

### 🌫️ Noise Simulation Module
Add different types of artificial noise to images:
- Gaussian Noise (controlled by σ)
- Uniform Noise
- Salt & Pepper Noise

### 🔧 Image Filtering Engine
Reduce noise using classical filters:
- Average Filter
- Gaussian Filter
- Median Filter  
Supports multiple kernel sizes
<img width="1392" height="898" alt="image" src="https://github.com/user-attachments/assets/70b5f82e-4913-4507-b765-98e85d1a1db0" />

---

### 🧭 Edge Detection Toolkit
Extract edges using multiple operators:
- Sobel (X & Y directions)
- Prewitt
- Roberts
- Canny Edge Detection (OpenCV integrated)
<img width="1403" height="928" alt="image" src="https://github.com/user-attachments/assets/41024edf-963c-4c7a-8570-d7de3ae7fe33" />

---

### 📊 Histogram Analysis Panel
Visualize image intensity distribution:
- Histogram plotting
- PDF (Probability Distribution Function)
- CDF (Cumulative Distribution Function)
- Separate histograms for R, G, and B channels

### ⚖️ Image Enhancement Tools
Improve image quality using:
- Histogram Equalization
- Intensity Normalization

Create hybrid images by combining:
- Low-frequency components of one image
- High-frequency components of another image
